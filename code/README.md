# Readme
### the folder is the code of **HACo-Det: A Study Towards Fine-Grained Machine-Generated Text Detection under Human-AI Coauthoring**

### The following is the explain of each file:

### dataset_construct.py: code for generating dataset

### dataset_filter_fill.py: code for human check

### dataset_preprocess.py: code for preprocess the raw text

### feature_extraction_model.py: code for calculating metrics for dataset

### length_distribution.py: code for length distribution analysis

### sequence_similarity.py: code for sequence similarity analysis

### metric_plot.py: code for plot the metric of dataset
